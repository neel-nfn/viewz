# Viewz — YouTube Studio OS

## Headline
Run your YouTube channels like a production studio — from research to publish.

## Description
Viewz is a complete production operating system for YouTube creators and teams. Manage your entire content pipeline with AI-powered SEO optimization, team collaboration, and analytics-driven insights.

### Key Features

**Content Pipeline**
- Kanban board for task management (Research → Script → Edit → Publish → Archive)
- Drag-and-drop task organization
- Task drawer with comments, attachments, and activity timeline

**AI SEO Assistant**
- Generate optimized titles, descriptions, and tags using Google Gemini
- Content Doctor diagnostics with SEO score
- Cost tracking for AI usage

**Research & Outliers**
- Score research ideas based on analytics data
- Outlier detection for high-performing content ideas
- Chrome Extension for quick research scoring on YouTube

**Team Collaboration**
- Role-based access (Admin, Manager, Writer, Editor)
- Comment threads with @mentions
- File attachments (scripts, thumbnails)
- Notification system

**Analytics**
- Channel overview dashboard
- Tasks per stage tracking
- Overdue task alerts
- 24-hour activity metrics

**Multi-Channel Support**
- Manage multiple YouTube channels
- Channel-specific pipelines
- Brand customization per channel
